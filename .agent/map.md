# Architecture Map

## Directory Structure

```
src/
  components/  ← UI elements and layout views (e.g. PeriodDetail)
  data/        ← Raw data definitions (historical periods, definitions)
  pages/       ← Page-level components
  types/       ← TypeScript definitions
  App.tsx      ← Main application layout and router setup
  index.css    ← Tailwind definitions and custom styles
```
